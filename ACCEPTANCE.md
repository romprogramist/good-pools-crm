## Что специально НЕ проверяем сейчас

- **SMTP/инвайт-email** — отложен на боевой деплой (на тесте не нужен).
- **X-Accel-Redirect** для защищённой раздачи uploads — оптимизация для боевого, на тесте файлы раздаются через app (медленнее, но работает).
- **Боевой домен компании** — будет отдельным микроэтапом после фидбека по тесту.
- **Бэкап uploads** — на боевом включим. Сейчас бэкапится только БД (ежедневно в 02:00, ротация 30 дней).
